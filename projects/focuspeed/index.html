Project Detail

FocuSpeed

Adaptive audio playback driven by real-time EEG-based focus estimation.

HCI EEG Biosignals Adaptive System

Purpose

Build an adaptive audio playback system that estimates a listener’s focus level from EEG signals in real time and dynamically tunes playback speed to reduce missed information and sustain engagement.

Significance

By linking cognitive state estimation to content delivery, the system aims to improve learning efficacy and create more human-aware media experiences for education and productivity.

Publications & Awards

  • Poster & Demo: Human Interface Symposium 2025 Student Contest (SICHI2025)
  • Encouragement Award for “FocuSpeed: EEG-driven adaptive audio playback”

My Role

  • Led EEG signal processing pipeline and focus estimation logic.
  • Implemented adaptive playback controller and UX flow.
  • Prepared demo assets and on-site presentation materials.
